Sesión 3 Padlet Vacacional

Comandos básicos


GIT INIT
Inicializa un nuevo repositorio Git en la carpeta actual.
GIT ADD
Agrega cambios al área de preparación (staging area) para que estén listos para ser confirmados en el próximo commit.
GIT COMMIT
Confirma los cambios en el repositorio. Crea una instantánea de los cambios que se agregaron al área de preparación (staging area).
GIT STATUS
Muestra el estado actual del repositorio, incluyendo los cambios pendientes, los archivos sin seguimiento y los archivos modificados.
GIT LOG
Muestra un registro de los commits realizados en el repositorio.
FONT SIZE
Cambia el tamaño de la fuente en la terminal.
COLOR
Cambia el color del texto en la terminal.
LS -LA
Muestra los archivos y carpetas en la carpeta actual, incluyendo los archivos y carpetas ocultos.
